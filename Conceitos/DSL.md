
- **Definição:** É uma das tecnologias tradicionais de rede de acesso residencial. Ela permite a transmissão de dados em alta velocidade através das linhas telefônicas de cobre já existentes, sem interferir nas chamadas de voz normais.
    
- **Como funciona:** A linha telefônica residencial conecta-se a um modem DSL, que converte os dados em sinais de alta frequência. Do lado da operadora, esses sinais chegam a um equipamento chamado **DSLAM (DSL Access Multiplexer)**, que separa o tráfego de voz (que vai para a rede telefônica) do tráfego de dados (que vai para a internet).
    
- **Característica:** Geralmente é _assimétrica_ (ADSL), o que significa que a velocidade de download (receber dados) é significativamente maior do que a velocidade de upload (enviar dados).
