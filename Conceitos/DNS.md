O DNS ( Domain Name System ) é basicamente uma lista de contatos. 

Basicamente o computador não entender o google.com, eles se comunicam por [[IP]]
Quando vc digita um link no navegador:

1. Seu pc pergunta para o servidor DNS qual é o IP do site X
2. o DNS procura no banco e responde o IP
3. Seu pc recebe o ip e aí sim consegue se conectar com o site.
